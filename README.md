# Realogy Holdings (realogy-holdings)
Realogy Holdings (now Anywhere Real Estate) is a global provider of residential real estate services with brands including Better Homes and Gardens Real Estate, Century 21, Coldwell Banker, Corcoran, ERA, and Sotheby's International Realty. Realogy Holdings does not currently publish a public developer API or developer portal; integrations are managed through brand-specific technology platforms and partner programs.
